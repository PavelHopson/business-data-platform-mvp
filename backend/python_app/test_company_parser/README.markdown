# Парсер данных российских компаний

Парсер для сбора ключевой информации о российских компаниях из разных источников.На данном этапе источник только один - "list-org.com".

## Описание

Этот проект собирает данные о российских компаниях с и формирует единый `output.json` файл с ключевыми бизнес-метриками.

## Источники данных

  * **list-org.com** - Финансы, руководители, контакты, ОГРН, учредители, вид деятельности, судебные дела.

## Установка

  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Запуск

Парсер поддерживает 3 режима запуска из командной строки:

1.  **Парсинг одного ИНН** (результат выводится в консоль и сохраняется в файл):

    ```bash
    python main.py --inn 7707083893
    ```

2.  **Парсинг нескольких ИНН** (результаты сохраняются в файл):

    ```bash
    python main.py --inn 7707083893,7736207543,9703073496
    ```

3.  **Парсинг из файла** (ИНН должны быть в файле `inns.txt` по одному на строку):

    ```bash
    python main.py --file inns.txt
    ```

    *Можно также указать свой выходной файл:*

    ```bash
    python main.py --file inns.txt --output my_results.json
    ```

## Результаты

После выполнения парсинга в директории `output/` (или по указанному вами пути) будет создан файл `output.json` (по умолчанию), содержащий список JSON-объектов успешно спарсенных компаний.

## Особенности

  * **Синхронный режим**: Использует `requests` для последовательной, но надежной загрузки.
  * **Устойчивость к блокировкам**: Применяются случайные задержки между запросами и ротация User-Agent (`fake-useragent`).
  * **Модульность**: Инфраструктура готова для добавления новых парсеров.
  * **Детальное логирование**: Весь процесс логируется в консоль и в файл `parser.log`.
